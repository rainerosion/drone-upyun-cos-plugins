# drone plugins for upyun cos

## 参数说明
| 参数             | 说明       | 备注     |
| ---------------- |----------|--------|
| up_operator      | 操作员名称    ||
| up_password      | 操作员密码    ||
| up_bucket        | bucket名称 | 服务名称   |
| local_base_path  | 本地路径     | 文件或文件夹 |
| remote_base_path | 对象存储路径   | 文件夹    |
